javascript introducion

JavaScript (JS) es un lenguaje de programación utilizado principalmente para crear páginas web interactivas. Se ejecuta en el navegador del usuario, lo que permite que las páginas web respondan a las acciones del usuario, como clics, desplazamientos o entradas de texto. Además de ser utilizado en el desarrollo web, JavaScript se emplea también para desarrollar aplicaciones móviles, videojuegos, e incluso en el lado del servidor a través de entornos como Node.js.

Una de las características más importantes de JavaScript es su capacidad para manipular el contenido de una página web en tiempo real sin necesidad de recargarla. Esto hace que las experiencias en línea sean mucho más dinámicas y rápidas. En resumen, JavaScript es esencial para hacer que las páginas web sean interactivas, funcionales y modernas.

    • javascript estructura

    JavaScript (JS) es un lenguaje de programación interpretado que se utiliza principalmente para desarrollar páginas web dinámicas. Su estructura se basa en el uso de variables, operadores, funciones, estructuras de control (como condicionales y bucles) y objetos. Los programas en JS pueden manipular el DOM (Document Object Model) de una página web, lo que permite actualizar dinámicamente el contenido de una página sin necesidad de recargarla.

    Además, JS es un lenguaje orientado a objetos y basado en prototipos, lo que permite crear objetos y heredar características de otros objetos. Las funciones también son fundamentales en JS, pudiendo ser utilizadas como objetos de primera clase.

    let a = 5; let b = 10; let resultado; resultado = a + b; if (resultado > 10) { console.log("El resultado es mayor que 10");}; else { console.log("El resultado es menor a 10");};

    • javascript tipos de datos

    En JavaScript, los tipos de datos se dividen en dos categorías principales: tipos primitivos y tipos de referencia. Los tipos primitivos incluyen números (que pueden ser enteros o decimales), cadenas de texto (strings), booleanos (valores true o false), null (un valor especial que representa la ausencia de un valor), undefined (un valor que se asigna automáticamente a las variables no inicializadas) y símbolos (un valor único e inmutable usado para identificadores).

    Además, existe el BigInt, que permite trabajar con enteros de tamaño arbitrario. Los tipos de referencia son objetos, como objetos literales, arrays, funciones y fechas, que se almacenan por referencia, lo que significa que múltiples variables pueden hacer referencia a la misma estructura de datos. A diferencia de los tipos primitivos, los tipos de referencia no se copian al asignarlos a nuevas variables, sino que ambas variables apuntan a la misma ubicación en memoria.

    let numero = 3; console.log(typeof numero); //con este operador se mira el tipo de dato

    • javascript variables

    En JavaScript, las variables se utilizan para almacenar datos y se pueden declarar utilizando las palabras clave var, let y const. var es la forma más antigua y tiene un alcance global o de función, lo que puede generar problemas de ambigüedad. let permite declarar variables con un alcance de bloque, lo que ayuda a evitar esos problemas, y se puede reasignar su valor.

    const, por su parte, también tiene un alcance de bloque, pero no permite la reasignación de su valor, lo que la hace ideal para constantes. Estas tres formas definen cómo se maneja el alcance y la reasignación de las variables en un programa.

    var x = 10; let y = 20; const z = 30;

    • javascript dom

    JavaScript, junto con el DOM (Document Object Model), permite interactuar y modificar el contenido y estructura de una página web de manera dinámica. A través del DOM, se pueden seleccionar elementos HTML, cambiar sus atributos, agregar eventos, y actualizar el contenido sin tener que recargar la página.

    Por ejemplo, usando document.getElementById(), se puede obtener un elemento y modificar su texto o estilo. Esta capacidad de manipulación en tiempo real es clave para crear páginas web interactivas y dinámicas. El DOM convierte a JavaScript en una herramienta poderosa para el desarrollo web.

    const cambiarTitulo = () => { document.getElementById("titulo").textContent = "¡Título cambiado!";}; document.getElementById("boton").addEventListener("click", cambiarTitulo);
